

## FormDialogMaker

表单对话框场景

```javascript
<form-dialog-maker 
    :dialog="dialog"
    :form="form"
    :api="api" 
/>
```
----

#### 自定义属性
|属性名称| 属性含义| 类型|
- | :-: | :-: | 
form | 表单配置 |  Object|
dialog | 对话框配置 | Object|
api | 提交接口 | Object.Promsie |

#### element-ui属性

除了上述属性外，其他属性和elment-ui一致，属性书写方式形如`minWidth`或`min-width`
